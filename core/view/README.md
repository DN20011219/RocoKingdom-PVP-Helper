# 可视化

本模块负责全部功能的可视化能力。

首先，本模块使用React或TS或JS或Vue的前端技术栈，并且将其封装为一个无边框的独立软件。

在此基础上，VIEW 分为几个主要功能区域：

1. 菜单区，固定在本软件最上方，供给用户选择功能

2. 内容区，除了上方菜单外，软件下方是内容区，根据不同的选定功能显示不同的内容。

这个软件的高度正好占满屏幕，宽度约为屏幕的1/6，位置固定在屏幕最右侧。


你需要做的：

1. 搭建整体框架，并完成第一阶段的打包发布和运行，让我能看到这个软件在屏幕右侧运行起来。

2. 内容区要尽可能支持可替换化和灵活化，因为我有很多不同的模块需要在这里显示完全不同的内容。
